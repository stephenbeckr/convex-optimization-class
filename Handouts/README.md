# Handouts for Convex Optimization Class

- `FirmlyNonexpansive.pdf` discusses properties of operators: (firmly) non-expansive, cocoercive, monotone, coercive.
- `FixedPtTheorems.pdf` puts Banach-Picard into context, mentioning Brouwer, Kakutani and Schauder theorems, as well as Schaefer, Caristi, etc.
- `StrongConvexityLipschitz.pdf` are about inequalities (using strong convexity and strong smoothness), a newer version of `SubOptimalityBounds.pdf`
- `SubgradientDescent.pdf` is a unified analysis of varous descent methods
- `SubOptimalityBounds.pdf` are about inequalities (using strong convexity and strong smoothness)
